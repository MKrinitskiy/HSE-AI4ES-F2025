# Домашнее задание №1  
## Вывод функции потерь линейной регрессии в предположении о распределении шума по Лапласу

Дедлайн: **03.10.2025**

### Цель задания
На лекции мы получили вывод стандартной функции потерь линейной регрессии (сумма квадратов ошибок) из предположения о нормальности распределения целевой переменной.  
В этом задании требуется **выполнить полностью аналогичный вывод**, но при другом предположении о распределении.

---

## Задание
Повторите шаг за шагом вывод, который был разобран на занятии, **заменив предположение о нормальном распределении на распределение Лапласа**.

Необходимо:

1. **Записать правдоподобие одного объекта** для выбранной модели распределения.  
2. **Записать правдоподобие всей выборки**, используя предположение независимости.  
3. **Переход к лог-правдоподобию**, используя свойства логарифма.  
4. **Отбросить константы**, не влияющие на положение оптимума.  
5. **Перейти от задачи максимизации правдоподобия к задаче минимизации функции потерь**.  
6. **Получить итоговую форму функции потерь**, соответствующую распределению Лапласа.

- Стиль — аккуратное, структурированное изложение, повторяющее ход вывода с лекции.
